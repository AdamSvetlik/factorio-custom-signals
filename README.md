# Custom Signals

A Factorio 2.0 mod that adds extra **virtual signals** for the circuit network.

## v1 — Train station signals

| Signal | Use |
| --- | --- |
| **Requester Station** | Mark stations that request items |
| **Provider Station** | Mark stations that provide items |
| **Depot Station** | Mark depot / parking stations |
| **Refuel Station** | Mark stations where trains refuel |

Use them on train stops, constant combinators, and circuit wires to label and
manage your rail network at a glance.

## Requesting new signals

This mod is meant to grow. If you need a signal that doesn't exist yet,
**open an issue** on this repository describing what you want and it may be
added in a future version.
